# Feature Showcase — Troibits Internship Task

This project is a **React + TypeScript + Tailwind CSS** web application built for the **Troibits Internship Sample Task**.  
It demonstrates a **Feature Showcase Section** with smooth scrolling, sticky behavior, and responsive UI.

---

## 🚀 Tech Stack

- ⚛️ **React (Vite + TypeScript)**
- 🎨 **Tailwind CSS**
- 🎞️ **Framer Motion** — for scroll-based animations
- 💡 **Responsive Design** — optimized for desktop, tablet, and mobile
